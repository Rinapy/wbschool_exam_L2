Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
2
1
Функция test() возвращает переменную x, увеличивая ее значение на 1 с помощью defer. Изначально значение x устанавливается равным 1, затем вызывается оператор defer, который увеличивает x на 1, в результате чего test() возвращает 2.
Функция anotherTest() устанавливает значение x равным 1, затем в блоке defer увеличивает его на 1, но при этом возвращает исходное значение x, которое равно 1.
Оператор defer позволяет отложить вызов функции до тех пор, пока не завершится текущая функция. При этом вызовы defer выполняются в порядке, обратном порядку их объявления. То есть последний defer вызывается первым, затем предпоследний вызывается следующим и так далее.
В случае функции test() оператор defer увеличивает значение x после того, как оно было присвоено и перед тем, как оно будет возвращено. В функции anotherTest() оператор defer также увеличивает значение x, но в этом случае оно не влияет на возвращаемое значение, так как оно уже было вернуто до вызова defer.

```
